================================================================================
FAST SETUP (3 STEPS)
================================================================================

1. WINDOWS USERS - Run setup script:
   Double-click: setup.bat
   
   LINUX/MAC USERS - Run setup script:
   Terminal: bash setup.sh

2. Wait for setup to complete (3-5 minutes)

3. When prompted, create your admin account:
   - Enter username (e.g., admin)
   - Enter email
   - Enter password
   - Confirm password

That's it! Server will be ready to start.

================================================================================
START THE WEBSITE
================================================================================

Open Terminal/Command Prompt in project folder and run:

    python manage.py runserver

Visit in your browser:
    http://127.0.0.1:8000/

Admin panel:
    http://127.0.0.1:8000/admin/

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Port 8000 already in use
SOLUTION: python manage.py runserver 8001

PROBLEM: Can't login to admin
SOLUTION: Make sure you created superuser with: python manage.py createsuperuser

PROBLEM: No products showing
SOLUTION: Load sample products: python manage.py load_sample_products

PROBLEM: Images not displaying
SOLUTION: Make sure media folder exists and has write permissions
