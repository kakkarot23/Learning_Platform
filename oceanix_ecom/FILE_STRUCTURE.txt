================================================================================
ROOT LEVEL FILES
================================================================================

📄 manage.py
   - Django management tool
   - Use to run migrations, create superuser, load data, start server
   - Main entry point for Django commands

📄 requirements.txt
   - Python dependencies list
   - Includes: Django 4.2, Pillow, python-decouple
   - Install with: pip install -r requirements.txt

📄 .gitignore
   - Git ignore rules
   - Excludes: database, caches, virtual environment, logs

📄 README.md
   - Complete project documentation
   - Installation, features, deployment, troubleshooting
   - Read first for full overview

📄 SETUP_GUIDE.txt
   - Step-by-step installation instructions
   - Detailed walkthrough of entire setup process
   - Common issues and fixes

📄 FEATURES.md
   - Complete feature documentation
   - User workflows, admin features, technical details
   - API endpoints and data models

📄 QUICKSTART.txt
   - Quick reference guide
   - Fast setup instructions
   - Common commands and troubleshooting

📄 setup.bat
   - Automated setup script for Windows
   - Creates virtual environment, installs dependencies, runs migrations
   - Double-click to run automatic setup

📄 setup.sh
   - Automated setup script for Linux/Mac
   - Same functionality as setup.bat
   - Run with: bash setup.sh
