TROUBLESHOOTING IF ANY BOX IS UNCHECKED
================================================================================

If any item is unchecked, see:
1. SETUP_GUIDE.txt for installation issues
2. README.md for configuration issues
3. FEATURES.md for functionality questions
4. Common commands in QUICKSTART.txt

Common Issues:
- No products showing → Run: python manage.py load_sample_products
- Admin styles missing → Run: python manage.py collectstatic --noinput
- Database errors → Run: python manage.py flush && python manage.py migrate
- Port in use → Run: python manage.py runserver 8001
